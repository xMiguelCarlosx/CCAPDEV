1. Download all the files and run it in VS CODE 
2. Run MONGODB and copy the string "mongodb://localhost:27017". It will also show the sample datas

3. Run the "index.js" in the terminal by typing "npm start". This should work by showing:
> MCO3@1.0.1 start
> node index      

Connected to MongoDB.
Express app now listening...

4. Type on the searchbar of the website "localhost:3000". This should show the Welcome Page with login and register buttons
5. Register if you don't have an account
    NOTE:
    Registering an existing account will show "Registration failed. The user already exists."

6. Logging in will direct you to the homepage that will allow you to reserve slots by going to the navigation bar on the left side
